# Cloud Integrated Vehicle Tracking & Protection System

## Description
This project is an ESP32-based smart vehicle security and tracking system using GPS, RFID, OLED display, relay module, and IoT cloud integration.

## Technologies Used
- ESP32
- GPS NEO-6M
- RFID MFRC522
- OLED SSD1306
- ThingSpeak IoT
- Relay Module

## Features
- Real-time vehicle tracking
- RFID-based authentication
- Cloud monitoring
- Vehicle protection system
- OLED status display

## Components
- ESP32
- GPS Module
- RFID Reader
- OLED Display
- Relay
- Buzzer
